# Weather App Project

A simple web application that allows users to check weather information for any city or based on their current location using an external Weather API.

> Note: Automatic device location detection depends on browser network-based services and may not work reliably in some regions. In such cases, users can search for a city manually.

You can try the project online [here](https://weather-app-seven-theta-ksfkd8q7ig.vercel.app/)

## Features

- Search weather information by city name
- Get current weather using device location
- Display temperature, humidity, and weather conditions
- Loading indicator while fetching data
- Error messages for invalid city names or failed requests
- Responsive design for different screen sizes

## Technologies Used

- HTML
- CSS
- JavaScript
- Weather API

## How to Use

Follow these steps to run the project locally:

1. Download the project  
   Click the green "Code" button on GitHub and choose "Download ZIP", then extract the folder.

2. Open the project  
   Navigate to the extracted folder and open the index.html file in your browser.

3. Use the Weather App
   - Search for a city to view its weather details.
   - Or use the Get Device Location option to fetch weather data based on your current location.

Or, you can try the project online [here](https://weather-app-seven-theta-ksfkd8q7ig.vercel.app/)
